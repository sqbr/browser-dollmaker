About this Dollmaker

This is a dollmaker for creating custom portraits and sprites for the game Stardew Valley. It is currently a very early alpha build and super unreliable! Saves made now will definitely break in later releases.

Please let me know if you hit any bugs or if there's any extra features you'd like, though I make no guarantees. I've never actually made a custom NPC myself but there's a lot of information about it on the wiki.

You can see my other mods and contact me at my Nexus page.

Planned Features

  • Proper image export
  • 'Randomise' buttons
  • A tutorial
  • Pre-made save files for all the main NPCs
  • Wheelchair option

Known Bugs

  • Occasional flicker while image is updating
  • Colours sometimes take a second or two to 'catch up' to the right value.
  • Import and right-click-save don't work on all browsers. Chrome and Firefox seem to be ok.
